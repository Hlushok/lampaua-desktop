# Резервна копія та перенесення налаштувань

У застосунку є два способи перенести або зберегти налаштування.

## Через ID і PIN

Цей спосіб зручний, коли треба перенести налаштування на інший пристрій.

1. На старому пристрої відкрийте **Налаштування застосунку**.
2. Перейдіть у **Резервна копія / перенесення**.
3. Натисніть **Створити ID і PIN**.
4. Запишіть показані **ID експорту** та **PIN-код**.
5. На новому пристрої відкрийте той самий розділ.
6. Натисніть **Ввести ID і PIN**.
7. Введіть ID експорту та PIN-код.

Код зберігається на сервері 1 годину. Дані перед відправкою шифруються PIN-кодом.

## Через файл

Цей спосіб зручний, коли треба зберегти копію на комп'ютері.

1. Відкрийте **Налаштування застосунку**.
2. Перейдіть у **Резервна копія / перенесення**.
3. Натисніть **Зберегти файл**, щоб створити файл резервної копії.
4. Натисніть **Відновити з файлу**, щоб імпортувати налаштування з такого файлу.

Файл можна перенести вручну на інший пристрій і відновити там.
