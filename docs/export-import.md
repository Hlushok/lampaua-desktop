# Экспорт и импорт настроек

Данный функционал позволяет сохранять и восстанавливать настройки приложения, а также переносить их между устройствами
или сборками.

Навигация:
[Экспорт настроек](#-экспорт-настроек) | [Импорт настроек](#-импорт-настроек) | [Формат файла](#формат-файла)

---

Если вы используете не эту версию приложения, то вам нужно установить расширение: https://lampa.kolovatoff.ru/ei.js

> ⚠️ Внимание! Вы можете экспортировать настройки через расширение в любой лампе, и импортировать также через расширение в любой лампе. \
> ❗ Важно, в эту сборку не нужно устанавливать расширение, весь функционал есть в разделе "Настройки приложения"

## 🔽 Экспорт настроек

### Облако

> ⚠️ Внимание! Ваши копии передаются на сервер в зашифрованном виде и хранятся 1 час.

|     | Эта сборка                                                                      | Через расширение                                                                |
| --- | ------------------------------------------------------------------------------- | ------------------------------------------------------------------------------- |
| 1   | Нажмите на кнопку настроек в Лампе, перейдите в раздел **Настройки приложения** | Нажмите на кнопку настроек в Лампе, перейдите в раздел **Экспорт/Импорт (Кфф)** |
| 2   | Выберите пункт **"Экспорт/Импорт настроек"**                                    |                                                                                 |
| 3   | Нажмите Экспорт в облако                                                        | Нажмите Экспорт в облако                                                        |
| 4   | Вы получите ID и PIN для расшифровки копии                                      | Вы получите ID и PIN для расшифровки копии                                      |

### Локально

> ⚠️ Внимание! Файл **может содержать** персональные данные аккаунтов или пароли. Поэтому не передавайте его кому-либо.\
> 🪧 Локальный способ работает только на ПК и в браузере.

|     | Эта сборка                                                                              | Через расширение                                                                        |
| --- | --------------------------------------------------------------------------------------- | --------------------------------------------------------------------------------------- |
| 1   | Нажмите на кнопку настроек в Лампе, перейдите в раздел **Настройки приложения**         | Нажмите на кнопку настроек в Лампе, перейдите в раздел **Экспорт/Импорт (Кфф)**         |
| 2   | Выберите пункт **"Экспорт/Импорт настроек"**                                            |                                                                                         |
| 3   | Нажмите Экспорт в файл                                                                  | Нажмите Экспорт в файл                                                                  |
| 4   | Укажите место сохранения файла (по-умолчанию называется: `lampaua-desktop-settings.json`) | Укажите место сохранения файла (по-умолчанию называется: `lampaua-desktop-settings.json`) |
| 5   | Нажмите **"Сохранить"**.                                                                | Нажмите **"Сохранить"**.                                                                |

Файл будет сохранён в формате `.json` и будет содержать:

- Ваши личные настройки десктопного приложения (URL Lampa, полноэкранный режим и т.п.).
- Настройки самой лампы (аккаунт, закладки и т.п.).

## Формат файла

Файл настроек — это JSON со следующей структурой:

```json lines
{
  "appVersion": "0.0.0",
  "dateCreated": "2026-02-03T18:57:43.964Z",
  "app": {},
  "lampa": {}
}
```

> 🔍 Поле `app` содержит настройки десктопного приложения (например: размер окна, URL Lampa, параметры обновления).  
> 📦 Поле `lampa` содержит данные самой Lampa (аккаунт, закладки, история, настройки интерфейса и т.д.).
